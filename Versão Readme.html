INTRODUÇÃO

A partir daqui se iniciam as aulas a respeito de HTML5 e suas possibilidades, essas características poderam ser abordadas livremente em um ambiente desafiador e de aspecto prático.

Vamos apontar as características de cada um dos tags principais que poderam ser utilizados para desenvolvimento de uma página vinculada a internet.

Lembrando que existem diversas editores que podem ser utilizados como: NOTEPAD++, BLOCO DE NOTAS, VISUALCODE E SIMILARES. Eu particularmente prefiro editar a partir do notepad!

    TAGS
  • head = Define a área reservada ao cabeçalho.
  • title = Insere o Título exibido na aba de notificação do navegador, utilizado dentro do marcador head.
  • body = Define o conteúdo visual a ser exibido (apresentado) pelo navegador, compreendendo textos, imagens e demais arquivos de mídia.
  • strong = Exibe caracteres em negrito.
  • em = Exibe caracteres em itálico.
  • p = parágrafo.
  • br = Define a quebra de linha.
  • h1 a h6 = Definem o tamanho da fonte a ser exibido em tela.
  • ul = Cria uma lista não ordenada (unordered list).
  • li = Define itens da lista.
  • ol = Define uma lista ordenada (ordered list).

Seguem abaixo exemplos das diferentes formas de lista.
